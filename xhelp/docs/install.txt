INSTALL/UNISTALL
=================

No special measures necessary, follow the standard installation process and extract the module folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual: http://goo.gl/adT2i

Unzip archive to 'modules' directory.
Install application using XOOPS module administration panel.
Adjust module preferences as necessary.
Add ticket departments (categories).
Setup XOOPS user accounts that represent helpdesk staff members.
Follow steps in "Block Styles" section of this document
All Set!!!
